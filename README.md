# 🎮 Projecte III Magrana Studios - Dos quarts

Aquest projecte és el resultat d'una feina a classe. Aquest videojoc consta d'una interfície que t'anirà guiant per escollir els nivells o mode de joc.

## 📁 Estructura del projecte

Dins del projecte a **Unity**, les carpetes estan separades en:
/**Assets** 
- **`Scripts/`** - Conté tots els codis font del joc.
- **`Scenes/`** - Aquí es troben les diferents escenes del joc (menú principal, nivells, etc.).
- **`Prefabs/`** - Objectes reutilitzables del joc.
- **`Art/`** - Recursos gràfics com sprites, textures i animacions.
- **`Audio/`** - Sons, música i efectes d'àudio.
- **`Animations/`** - Controladors i clips d'animació.
